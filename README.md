# islands-bitmap
This app will randomize width x height matrix and will draw it on the screen. <br>
Furthermore, you can find how many islands the random matrix has by pressing solve.<br>

## Stack

### Backend

- API
  - NodeJS
  - Express
- Proxy
  - NGINX

### Frontend

- Web
  - React

### Deployment

- Technologies
  - Docker
  - Docker compose

## Setup and Running

- Prerequisites

  - Node (`v10.x`)
